ShowerScribe (SE101 Project)
  • Skills used: Python, Flask, HTML, CSS, APIs, test case analysis, teamwork
  • Duration: 2 months
Link to presentation video of the project: ShowerScribe

This is a project that I collaborated with 5 other classmates to complete, which uses Raspberry Pi as main hardware components and Python as main programming language.

    Some features include:
  • Easy use: it can record your voice and shower thoughts upon a push of a button.
  • Voice transcription to transcribe the recordings, which is what I mainly worked on.
  • LLM services and semantic search to categorize the recordings
  • Waterproof, can be installed easily in the shower.

One of the main challenges that I faced during the project is to consider any post-audio modification after recording, since there could be some background noise of the shower that can cause the AI model to not be able to filter out the voice accurately. Another challenge is to figure out how to link everything together using a conductor, for auto startup.
